# CLI ToDo (C++)

Simple command-line ToDo application written in C++.

## Features
- Add tasks
- List tasks
- Help menu
- Case-insensitive commands

## Commands
- add
- list
- help
- exit

## Run

bash
g++ main.cpp -o todo
./todo
